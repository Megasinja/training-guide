アルゴリズム早見表

Partitioning

K-means

重心との距離が最小になるように点を割り当てる反復法。球状クラスタに強い。

J = Σ_i ||x_i - μ_{c_i}||² c_i = argmin_k ||x_i - μ_k||² μ_k = (1/|C_k|) Σ_{i∈C_k} x_i
  • 主パラメータ: クラスタ数 K
  • 注意点: 初期値依存、外れ値に弱い
Density-based

DBSCAN

密度連結でクラスタを見つける。外れ値をノイズとして扱える。

N_ε(p) = {q | dist(p,q) ≤ ε} core(p) ⇔ |N_ε(p)| ≥ MinPts cluster = density-reachable set
  • 主パラメータ: ε, MinPts
  • 注意点: 密度差が大きいデータは調整が難しい
Hierarchical

Agglomerative

各点を1クラスタとして開始し、近いクラスタ同士を順にマージ。

d_single(A,B) = min_{a∈A,b∈B} dist(a,b) d_complete(A,B) = max_{a∈A,b∈B} dist(a,b) d_average(A,B) = mean_{a∈A,b∈B} dist(a,b)
  • 主パラメータ: linkage, 目標クラスタ数 K
  • 注意点: データ数が増えると計算が重くなる
Probabilistic

GMM (EM)

ガウス混合分布を仮定し、責務とパラメータを交互更新。楕円クラスタに対応。

p(x)=Σ_k π_k N(x|μ_k,Σ_k) r_nk = π_k N(x_n)/Σ_j π_j N_j(x_n) μ_k, Σ_k, π_k を M-step で更新
  • 主パラメータ: 成分数 K、分散下限
  • 注意点: 局所解・初期値依存
Graph-based

Spectral

類似度グラフの固有空間へ埋め込み、埋め込み空間でK-meansを実行。

W_ij = exp(-||x_i-x_j||²/2σ²) S = D^{-1/2} W D^{-1/2} U = top-K eigenvectors, then K-means(U)
  • 主パラメータ: K、k近傍、σ
  • 注意点: 固有計算が重い
Mode-seeking

Mean Shift

密度勾配上昇でモードへ収束させ、収束先でクラスタ化。Kの事前指定不要。

m(x)=Σ_i K_h(x_i-x)x_i / Σ_i K_h(x_i-x) x ← m(x) 近いモードを統合してクラスタ化
  • 主パラメータ: bandwidth h
  • 注意点: 計算量が大きい
Density-based

OPTICS

処理順とreachability距離を出力し、閾値で多様な密度クラスタを抽出。

coreDist(p), reachDist(q) seed集合を最小reachDist優先で展開 reachability plot からクラスタ抽出
  • 主パラメータ: ε、MinPts、抽出閾値
  • 注意点: 可視化解釈が必要
Large-scale

BIRCH

CF要約を木構造で保持し、逐次挿入しながらマイクロクラスタを作る。

CF=(N,LS,SS), μ=LS/N R=sqrt(SS/N-||μ||²) R≤T なら吸収、超えれば新規CF
  • 主パラメータ: しきい値 T、分岐係数 B
  • 注意点: 要約段階の設計で品質が変化

推奨学習順: K-means → DBSCAN → 階層クラスタリング。 前者ほど直感的で、後者ほど表現力と設定の自由度が上がります。
