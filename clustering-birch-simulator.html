BIRCH クラスタリングシミュレータ（CF要約）

BIRCH はデータを1件ずつ挿入し、Clustering Feature (CF) で要約しながら マイクロクラスタを構築する大規模向け手法です。

CF entry CF = (N, LS, SS) N: 点数, LS: 座標和, SS: 二乗和 centroid μ = LS / N radius R = sqrt( SS/N - ||LS/N||² ) 挿入 最近CFへ追加後 R ≤ T なら吸収、超えるなら新規CF （本デモは overflow 時に近いCF同士を圧縮マージ）
パラメータ説明
`しきい値 T` はCF半径上限、`分岐係数 B` は保持するCF数の上限です。 Tを小さくすると細かい要約、Bを小さくすると圧縮マージが増えます。
