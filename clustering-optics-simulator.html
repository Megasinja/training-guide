OPTICS クラスタリングシミュレータ

OPTICSは密度ベース法で、点を処理順に並べた reachability プロットからクラスタ構造を読み取ります。 DBSCANと違い、密度の異なる構造も比較的扱いやすいです。

近傍 N_ε(p) = { q | dist(p,q) ≤ ε } core distance coreDist(p) = MinPts番目近傍までの距離 reachability update reachDist(q) = min( reachDist(q), max(coreDist(p), dist(p,q)) ) 処理順序を出力し、reachability閾値でクラスタ抽出
パラメータ説明
`ε` と `MinPts` は密度判定、`抽出閾値 τ` は reachability プロットを切る高さです。 τを下げるほど厳密な高密度クラスタのみ残ります。
